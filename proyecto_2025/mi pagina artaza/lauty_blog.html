Presentación del proyecto

Locoste es una tienda pensada para la gente joven y adulta de la ciudad de Salta que busca moda, identidad y productos originales. El objetivo principal fue crear una web fácil de usar, rápida y con diseño minimalista.

Diseño minimalista y simplificación

Durante el desarrollo, se simplificaron todas las páginas para que tengan solo lo esencial: colores neutros, estructura clara y sin elementos avanzados. Se eliminaron barras de navegación complejas, filtros y efectos para lograr una experiencia directa y agradable.

Página de ofertas

La sección de ofertas muestra los productos destacados en tarjetas simples, sin filtros ni navegación avanzada. El usuario puede ver rápidamente las promociones y precios especiales.

Login y formularios

La página de login y los formularios fueron diseñados para ser lo más sencillos posible, con campos básicos y un pie de página fijo. Se eliminaron funciones avanzadas para facilitar el acceso y registro.

Curriculum

La página de curriculum permite mostrar información personal y profesional de forma clara y visualmente atractiva, manteniendo la coherencia con el diseño general.

Identidad y comunidad

Locoste fue creada para que la gente de la ciudad de Salta se sienta identificada y parte de una comunidad. Cada sección de la web busca reflejar esa identidad local y el espíritu de la marca.
